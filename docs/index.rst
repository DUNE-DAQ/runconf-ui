`Runconf-UI`
=============

..container:: badges
    ..  image:: https://github.com/DUNE-DAQ/runconf-ui/actions/workflows/run_pytest.yml/badge.svg
        :target: https://github.com/DUNE-DAQ/runconf-ui/actions/workflows/run_pytest.yml
        :alt: pytest

`Runconf-UI` is a textual user interface for configuring and managing DAQ runs. It provides an intuitive interface for users to create, modify, and manage run configurations, making it easier to set up and execute DAQ runs effectively. The application is built using the Textual framework.
