Installation
====================

Runconf-UI is designed to be installed as part of the DUNE DAQ environment. 
Details for setting up the environment can be found `here <https://dune-daq-sw.readthedocs.io/en/latest/packages/daq-buildtools/>`

Once you have the environment set up, you can install runconf-ui either using `DBT`

.. code-block:: bash

    dbt-build

Or via `pip` from the source code

.. code-block:: bash

    pip install .
